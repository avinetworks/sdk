Avi ServiceEngineGroup Object API

##CLI## ``` - configure serviceenginegroup - show serviceenginegroup - redistribute serviceenginegroup ``` ##Examples## - **serviceenginegroup_example_1**: To create a SE group that is suitable for a production application with a HA policy of shared active/active with a minimum scale out of 2 SE ```json {u'min_scaleout_per_vs': 2, u'ha_mode': u'HA_MODE_SHARED_PAIR', u'name': u'production-group'} ``` - **serviceenginegroup_example_2**: To create a SE group that is of best effort category suitable for test deployments ```json {u'buffer_se': 0, u'ha_mode': u'HA_MODE_SHARED', u'max_se': 8, u'name': u'test-group'} ```
